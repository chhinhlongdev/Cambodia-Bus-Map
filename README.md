# Cambodia Bus Map | ផែនទីឡានក្រុង

An interactive web app for exploring bus routes across Cambodia.

## Features

- Interactive Leaflet map with all major bus routes
- Filter by company (Giant Ibis, Mekong Express, Capitol Tour, Virak Buntham, Local)
- Filter by bus type (Express, Local, Night Bus)
- Search routes by origin and destination
- Schedule viewer with departure times and seat availability
- Route detail modal with stops, amenities, and booking info
- Responsive design for mobile and desktop

## Usage

Open `index.html` in a browser — no build step required.

For best results, serve it via a local server (e.g. VS Code Live Server) so `data/routes.json` loads correctly.

## Tech Stack

- HTML / CSS / Vanilla JS
- [Leaflet.js](https://leafletjs.com/) for maps
- [Font Awesome](https://fontawesome.com/) for icons
- [Google Fonts](https://fonts.google.com/) — Roboto & Battambang
